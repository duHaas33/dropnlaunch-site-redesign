# Drop N Launch Site Redesign Prototype

Standalone static redesign prototype for the public Drop N Launch marketing site.

This intentionally does not touch `dropnlaunch-web` or the app code. Open `index.html` directly, or serve this folder with a tiny local server for browser testing.

Sources folded into the prototype:

- Live homepage content from `https://dropnlaunch.com`
- Current setup guide content from `/get-started`, `/garage-setup`, `/batting-cage-setup`, and `/field-setup`
- Current brand/app/social assets from the local `dropnlaunch-web/public` folder and the live site
